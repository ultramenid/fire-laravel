# MapBiomas Indonesia Fire - Website

### Technology stack

- Laravel
- Laravel-Livewire
- AlpineJS
- TailwindCss
- MySQL

### How to use

- clone this repository
- composer install & npm install
- cp .env.example .env
- php artisan key:generate
- php artisan migrate
- php artisan storage:link
- create user using `php artisan tinker`
